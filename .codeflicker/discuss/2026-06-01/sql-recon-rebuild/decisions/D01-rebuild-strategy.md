## 📋 背景

### 问题/需求
v1（HelloAgents-SQL-Reconciliation-Agent）经过详细技术挑战后，确认存在以下根本性问题：

1. **抽象层次错配**：用"认知架构"叙事包了脚本式 demo 内核
2. **形似神不似**：三层记忆没 consolidation、自进化没门槛、双层 Agent 不共享 context
3. **业余实现痕迹**：关键字黑名单防 SQL 注入、print + stdout tee 做可观测、反射魔法做工具注册
4. **无验收准绳**：没评测集，任何改动都是手感工程

mabohui 即将面试，需要在 6 周以上时间内对项目进行从零重构，以提升简历可信度和讲故事能力。

### 约束条件
- 个人项目，单人开发
- 必须可在面试场景被"严肃面试官"质询而不露馅
- 必须可在 GitHub 公开（README/架构图/演示均要可用）
- 不必考虑真实业务部署的合规和 SLA

## ✅ 最终决策

### 选定方案
**v2 重构策略**：
1. **定位**：面试简历升级，讲故事优先
2. **周期**：6.5 周完整 Stage 0-5
3. **框架**：LangGraph 替代自研 HelloAgents 编排层
4. **依赖**：允许引入 Qdrant（向量检索）、SQLite（结构化持久化）、Redis（cache，可选）、OpenTelemetry（observability）

### 决策理由

#### 为什么是"简历升级"而非"准生产"
- 6 周做不出真正的生产级 SQL Agent（缺少 SLO、灰度、限流、租户隔离等）
- 面试场景需要的是**可以被严肃质询的工程实践**，不是真上线
- 一旦定位为"简历"，所有取舍清晰：宁可少做一个 feature，也要把已有的做到能讲出来

#### 为什么 6.5 周完整版 > 3 周精装版
- 3 周做到 Stage 2 后停下，Memory/RAG 还是 v1 状态，**简历叙事不完整**
- 三大亮点（双层 Agent / Tool+RAG / 记忆+自进化）必须全部"形神兼备"
- Stage 4（Memory + Sandbox 自进化）是最有面试价值的部分，跳过损失最大

#### 为什么切 LangGraph 而非保留 HelloAgents
- 十年经验判断：面试官看到"自研框架"会怀疑**不会用工业级工具**
- LangGraph 提供 state checkpoint / 可视化 / 中断恢复 / 多 Agent 协作，**这些自研要再花 4 周**
- "我用 LangGraph + AgentContext 模式实现了可观测、可评测的 SQL Agent" 比 "我自研了 mini agent framework" 好讲 10 倍
- HelloAgents 不是丢掉，是退化为"我学习自研框架后切换到工业标准"的成长叙事

#### 为什么允许外部依赖
- Qdrant + SQLite + Redis + OTel 这套是 **2025 年 Agent 工程标准栈**
- 拒绝它们 = 拒绝主流工程实践
- 简历提到这些 = 面试官知道你有"工业实战意识"

### 预期效果

#### 面试维度
- 三大亮点全部"形神兼备"，被任何深度追问都能答
- 主动暴露"我从自研切到 LangGraph 的原因"，反向证明工程判断力
- 50 条 Golden Set + Eval-driven 改造 = 比 99% 个人项目专业

#### 技术维度
- 完整跑通 Plan → Act → Observe → Reflect 闭环（LangGraph State Machine）
- Self-Evolution 加入 Sandbox 验证，自进化第一次有 governance
- Memory 真正有 promotion / consolidation / 衰减
- RAG 升级 Hybrid + Rerank，达到生产级检索水准

## ❌ 被否决的方案

### 方案 A：1 周止血版
- **否决原因**：与 v1 拉不开差距，对简历提升有限
- **重新考虑条件**：如果用户面试日期突然提前到 1 周内

### 方案 B：保留 HelloAgents 自研框架
- **否决原因**：面试场景下"为什么不用 LangGraph"是必问题，自研框架答不上来
- **重新考虑条件**：用户简历核心叙事必须是"自研 Agent 框架作者"

### 方案 C：纯单文件零依赖
- **否决原因**：与"工程现代化"目标冲突，主流栈对接不上
- **重新考虑条件**：项目目标转为"嵌入式 / 边缘部署"

### 方案 D：3 周精装版
- **否决原因**：Memory + 自进化保留 v1 残留，三大亮点叙事仍有一个不完整
- **重新考虑条件**：时间窗口被压缩到 3 周
