# LeetCode 570: 至少有5名直接下属的经理

**难度**: medium
**LeetCode题号**: 570

## 题目解读
找出至少有五个直接下属的经理

## 表: `Employee_570`

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | — |
| name | TEXT | — |
| department | TEXT | — |
| managerId | INTEGER | — |

## 参考SQL（SQLite）
```sql
SELECT name FROM Employee_570 WHERE id IN (SELECT managerId FROM Employee_570 GROUP BY managerId HAVING COUNT(*) >= 5);
```

## MySQL原始SQL（供方言对比）
```sql
select name from Employee where id in (select managerId from Employee group by managerId having count(*)>=5);
```
