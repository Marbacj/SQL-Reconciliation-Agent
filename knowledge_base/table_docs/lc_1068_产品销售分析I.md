# LeetCode 1068: 产品销售分析I

**难度**: easy
**LeetCode题号**: 1068

## 题目解读
获取Sales表中所有sale_id对应的product_name以及该产品的所有year和price

## 表: `Sales`

| 字段 | 类型 | 说明 |
|------|------|------|
| sale_id | INTEGER | — |
| product_id | INTEGER | — |
| year | INTEGER | — |
| quantity | INTEGER | — |
| price | INTEGER | — |
| PRIMARY | KEY | — |

## 表: `Product`

| 字段 | 类型 | 说明 |
|------|------|------|
| product_id | INTEGER | — |
| product_name | TEXT | — |

## 参考SQL（SQLite）
```sql
SELECT t2.product_name, t1.year, t1.price FROM Sales AS t1 LEFT JOIN Product AS t2 ON t1.product_id = t2.product_id;
```

## MySQL原始SQL（供方言对比）
```sql
select t1.year,t1.price,t2.product_name from Sales as t1 left join Product as t2 on t1.product_id = t2.product_id;
```
