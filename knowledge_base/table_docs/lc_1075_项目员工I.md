# LeetCode 1075: 项目员工I

**难度**: easy
**LeetCode题号**: 1075

## 题目解读
查询每一个项目中员工的平均工作年限，精确到小数点后两位

## 表: `Project`

| 字段 | 类型 | 说明 |
|------|------|------|
| project_id | INTEGER | — |
| employee_id | INTEGER | — |
| PRIMARY | KEY | — |

## 表: `Employee_1075`

| 字段 | 类型 | 说明 |
|------|------|------|
| employee_id | INTEGER | — |
| name | TEXT | — |
| experience_years | INTEGER | — |

## 参考SQL（SQLite）
```sql
SELECT t1.project_id, ROUND(AVG(t2.experience_years), 2) AS average_years FROM Project AS t1 JOIN Employee_1075 AS t2 ON t1.employee_id = t2.employee_id GROUP BY project_id;
```

## MySQL原始SQL（供方言对比）

select t1.project_id, round(avg(t2.experience_years),2) as average_years from Project as t1 join Employee as t2 on t1.employee_id = t2.employee_id group by project_id;
