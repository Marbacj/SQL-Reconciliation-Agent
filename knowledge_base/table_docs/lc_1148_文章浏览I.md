# LeetCode 1148: 文章浏览I

**难度**: easy
**LeetCode题号**: 1148

## 题目解读
查询出所有浏览过自己文章的作者，结果按id升序排列

## 表: `Views`

| 字段 | 类型 | 说明 |
|------|------|------|
| article_id | INTEGER | — |
| author_id | INTEGER | — |
| viewer_id | INTEGER | — |
| view_date | TEXT | — |

## 参考SQL（SQLite）
```sql
SELECT DISTINCT author_id AS id FROM Views WHERE author_id = viewer_id ORDER BY id;
```

## MySQL原始SQL（供方言对比）
```sql
select distinct t1.author_id as id from Views as t1 where t1.author_id = t1.viewer_id order by id;
```
