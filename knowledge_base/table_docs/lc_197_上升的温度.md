# LeetCode 197: 上升的温度

**难度**: easy
**LeetCode题号**: 197

## 题目解读
找出与之前日期相比温度更高的所有日期的id

## 表: `Weather`

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | — |
| recordDate | TEXT | — |
| temperature | INTEGER | — |

## 参考SQL（SQLite）
```sql
SELECT a.id FROM Weather AS a JOIN Weather AS b ON julianday(a.recordDate) - julianday(b.recordDate) = 1 AND a.temperature > b.temperature;
```

## MySQL原始SQL（供方言对比）
```sql
select a.id from Weather as a, Weather as b where datediff(a.recordDate,b.recordDate) = 1 and a.Temperature > b.Temperature;
```

## 方言差异说明
- DATEDIFF → julianday() 差值计算
