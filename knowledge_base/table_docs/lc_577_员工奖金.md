# LeetCode 577: 员工奖金

**难度**: easy
**LeetCode题号**: 577

## 题目解读
报告每个奖金少于1000的员工的姓名和奖金数额

## 表: `Employee_577`

| 字段 | 类型 | 说明 |
|------|------|------|
| empId | INTEGER | — |
| name | TEXT | — |
| supervisor | INTEGER | — |
| salary | INTEGER | — |

## 表: `Bonus`

| 字段 | 类型 | 说明 |
|------|------|------|
| empId | INTEGER | — |
| bonus | INTEGER | — |

## 参考SQL（SQLite）
```sql
SELECT t1.name, t2.bonus FROM Employee_577 AS t1 LEFT JOIN Bonus AS t2 ON t1.empId = t2.empId WHERE t2.bonus IS NULL OR t2.bonus < 1000;
```

## MySQL原始SQL（供方言对比）
```sql
select t1.name,t2.bonus from Employee as t1 left join Bonus as t2 on t1.empId = t2.empId where t2.bonus is null or t2.bonus <1000;
```
