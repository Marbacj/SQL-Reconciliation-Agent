# LeetCode 620: 有趣的电影

**难度**: easy
**LeetCode题号**: 620

## 题目解读
找出所有影片描述为非boring的并且id为奇数的影片，按rating降序排列

## 表: `cinema`

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | — |
| movie | TEXT | — |
| description | TEXT | — |
| rating | REAL | — |

## 参考SQL（SQLite）
```sql
SELECT * FROM cinema WHERE description != 'boring' AND id % 2 != 0 ORDER BY rating DESC;
```

## MySQL原始SQL（供方言对比）
```sql
select * from cinema where description != 'boring' and id%2!=0 order by rating desc;
```

## 方言差异说明
- 格式规范化（大写、空格等）
