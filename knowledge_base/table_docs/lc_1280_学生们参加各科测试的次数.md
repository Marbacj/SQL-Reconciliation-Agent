# LeetCode 1280: 学生们参加各科测试的次数

**难度**: easy
**LeetCode题号**: 1280

## 题目解读
查询每个学生参加每一门科目测试的次数，结果按student_id和subject_name排序

## 表: `Students`

| 字段 | 类型 | 说明 |
|------|------|------|
| student_id | INTEGER | — |
| student_name | TEXT | — |

## 表: `Subjects`

| 字段 | 类型 | 说明 |
|------|------|------|
| subject_name | TEXT | — |

## 表: `Examinations`

| 字段 | 类型 | 说明 |
|------|------|------|
| student_id | INTEGER | — |
| subject_name | TEXT | — |

## 参考SQL（SQLite）
```sql
SELECT t1.student_id, t1.student_name, t2.subject_name, COUNT(t3.subject_name) AS attended_exams FROM Students AS t1 JOIN Subjects AS t2 LEFT JOIN Examinations AS t3 ON t1.student_id = t3.student_id AND t2.subject_name = t3.subject_name GROUP BY t1.student_id, t2.subject_name ORDER BY t1.student_id, t2.subject_name;
```

## MySQL原始SQL（供方言对比）
```sql
select t1.student_id, t1.student_name, t2.subject_name, count(t3.subject_name) as attended_exams from Students as t1 join Subjects as t2 left join Examinations as t3 on t1.student_id = t3.student_id and t2.subject_name = t3.subject_name group by t1.student_id, t2.subject_name order by t1.student_id, t2.subject_name;
```
