# LeetCode 584: 寻找用户推荐人

**难度**: easy
**LeetCode题号**: 584

## 题目解读
找出没有被 id=2 的客户推荐的客户的姓名

## 表: `Customer`

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | — |
| name | TEXT | — |
| referee_id | INTEGER | — |

## 参考SQL（SQLite）
```sql
SELECT name FROM Customer WHERE referee_id != 2 OR referee_id IS NULL;
```

## MySQL原始SQL（供方言对比）
```sql
select name from Customer where referee_id != 2 or referee_id is null;
```

## 方言差异说明
- 格式规范化（大写、空格等）
