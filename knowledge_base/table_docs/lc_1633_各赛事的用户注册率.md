# LeetCode 1633: 各赛事的用户注册率

**难度**: easy
**LeetCode题号**: 1633

## 题目解读
统计各赛事的用户注册百分率，保留两位小数，按percentage降序排序

## 表: `Users`

| 字段 | 类型 | 说明 |
|------|------|------|
| user_id | INTEGER | — |
| user_name | TEXT | — |

## 表: `Register`

| 字段 | 类型 | 说明 |
|------|------|------|
| contest_id | INTEGER | — |
| user_id | INTEGER | — |
| PRIMARY | KEY | — |

## 参考SQL（SQLite）
```sql
SELECT contest_id, ROUND(COUNT(user_id) * 100.0 / (SELECT COUNT(*) FROM Users), 2) AS percentage FROM Register GROUP BY contest_id ORDER BY percentage DESC, contest_id;
```

## MySQL原始SQL（供方言对比）
```sql
select contest_id, round(count(user_id)*100 / (select count(*) from Users),2) as percentage from Register group by contest_id order by percentage desc, contest_id;
```
