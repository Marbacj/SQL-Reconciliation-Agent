# LeetCode 1251: 平均售价

**难度**: easy
**LeetCode题号**: 1251

## 题目解读
查找每种产品的平均售价，四舍五入到小数点后两位，如果没有售出则平均售价为0

## 表: `Prices`

| 字段 | 类型 | 说明 |
|------|------|------|
| product_id | INTEGER | — |
| start_date | TEXT | — |
| end_date | TEXT | — |
| price | INTEGER | — |
| PRIMARY | KEY | — |

## 表: `UnitsSold`

| 字段 | 类型 | 说明 |
|------|------|------|
| product_id | INTEGER | — |
| purchase_date | TEXT | — |
| units | INTEGER | — |

## 参考SQL（SQLite）
```sql
SELECT a.product_id, ROUND(SUM(a.t_price) / SUM(a.units), 2) AS average_price FROM (SELECT p.product_id, p.price * u.units AS t_price, u.units AS units FROM Prices p JOIN UnitsSold u ON u.product_id = p.product_id WHERE u.purchase_date BETWEEN p.start_date AND p.end_date) AS a GROUP BY a.product_id;
```

## MySQL原始SQL（供方言对比）
```sql
SELECT a.product_id, ROUND(SUM(a.t_price)/SUM(a.units),2) AS average_price FROM (SELECT p.product_id, p.price*u.units AS t_price, u.units AS units FROM Prices p JOIN UnitsSold u ON u.product_id=p.product_id WHERE u.purchase_date BETWEEN p.start_date AND p.end_date) AS a GROUP BY a.product_id;
```
