# LeetCode 1661: 每台机器进程的平均运行时间

**难度**: easy
**LeetCode题号**: 1661

## 题目解读
计算每台机器完成一个进程任务的平均耗时，四舍五入保留3位小数

## 表: `Activity`

| 字段 | 类型 | 说明 |
|------|------|------|
| machine_id | INTEGER | — |
| process_id | INTEGER | — |
| activity_type | TEXT CHECK(activity_type IN ('start','end') | — |
| timestamp | REAL | — |

## 参考SQL（SQLite）
```sql
SELECT a.machine_id, ROUND(AVG(b.timestamp - a.timestamp), 3) AS processing_time FROM (SELECT * FROM Activity WHERE activity_type = 'start') AS a LEFT JOIN (SELECT * FROM Activity WHERE activity_type = 'end') AS b ON a.machine_id = b.machine_id AND a.process_id = b.process_id GROUP BY machine_id;
```

## MySQL原始SQL（供方言对比）
```sql
select a.machine_id, round(avg(b.timestamp - a.timestamp),3) as processing_time from (select * from Activity where activity_type = 'start') as a left join (select * from Activity where activity_type = 'end') as b on a.machine_id = b.machine_id and a.process_id = b.process_id group by machine_id;
```
