# LeetCode 1934: 确认率

**难度**: medium
**LeetCode题号**: 1934

## 题目解读
查找每个用户的确认率（confirmed消息数除以总请求数），四舍五入到两位小数，没有请求的用户确认率为0

## 表: `Signups`

| 字段 | 类型 | 说明 |
|------|------|------|
| user_id | INTEGER | — |
| time_stamp | TEXT | — |

## 表: `Confirmations`

| 字段 | 类型 | 说明 |
|------|------|------|
| user_id | INTEGER | — |
| time_stamp | TEXT | — |
| action | TEXT CHECK(action IN ('confirmed','timeout') | — |

## 参考SQL（SQLite）
```sql
SELECT T1.user_id, ROUND(COUNT(CASE WHEN T2.action = 'confirmed' THEN 1 ELSE NULL END) * 1.0 / COUNT(T2.user_id), 2) AS confirmation_rate FROM Signups AS T1 LEFT JOIN Confirmations AS T2 ON T1.user_id = T2.user_id GROUP BY T1.user_id;
```

## MySQL原始SQL（供方言对比）
```sql
select T1.user_id, round(count(if(T2.action = 'confirmed',true,null)) / count(*),2) as confirmation_rate from Signups as T1 left join Confirmations as T2 on T1.user_id = T2.user_id group by T1.user_id;
```
