# LeetCode 595: 大的国家

**难度**: easy
**LeetCode题号**: 595

## 题目解读
找出面积至少300万平方公里或人口至少2500万的大国的名称、人口和面积

## 表: `World`

| 字段 | 类型 | 说明 |
|------|------|------|
| name | TEXT | — |
| continent | TEXT | — |
| area | INTEGER | — |
| population | INTEGER | — |
| gdp | INTEGER | — |

## 参考SQL（SQLite）
```sql
SELECT name, population, area FROM World WHERE area >= 3000000 OR population >= 25000000;
```

## MySQL原始SQL（供方言对比）
```sql
select name,population,area from World where area >= 3000000 or population >=25000000;
```
