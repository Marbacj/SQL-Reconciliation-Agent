# LeetCode 1683: 无效的推文

**难度**: easy
**LeetCode题号**: 1683

## 题目解读
查询所有无效推文的编号，当推文内容字符数严格大于15时该推文无效

## 表: `Tweets`

| 字段 | 类型 | 说明 |
|------|------|------|
| tweet_id | INTEGER | — |
| content | TEXT | — |

## 参考SQL（SQLite）
```sql
SELECT tweet_id FROM Tweets WHERE LENGTH(content) > 15;
```

## MySQL原始SQL（供方言对比）
```sql
select tweet_id from Tweets where char_length(content) > 15;
```

## 方言差异说明
- CHAR_LENGTH() → LENGTH()
