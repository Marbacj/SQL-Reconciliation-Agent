# LeetCode 1378: 使用唯一标识码替换员工ID

**难度**: easy
**LeetCode题号**: 1378

## 题目解读
展示每位员工的唯一标识码，如果某位员工没有唯一标识码则使用null填充

## 表: `Employees`

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | — |
| name | TEXT | — |

## 表: `EmployeeUNI`

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | — |
| unique_id | INTEGER | — |
| PRIMARY | KEY | — |

## 参考SQL（SQLite）
```sql
SELECT t2.unique_id, t1.name FROM Employees AS t1 LEFT JOIN EmployeeUNI AS t2 ON t1.id = t2.id;
```

## MySQL原始SQL（供方言对比）
```sql
select t2.unique_id,t1.name from Employees as t1 left join EmployeeUNI as t2 on t1.id = t2.id;
```
