# LeetCode 1757: 可回收且低脂的产品

**难度**: easy
**LeetCode题号**: 1757

## 题目解读
找出既是低脂又是可回收的产品编号

## 表: `Products`

| 字段 | 类型 | 说明 |
|------|------|------|
| product_id | INTEGER | — |
| low_fats | TEXT CHECK(low_fats IN ('Y','N') | — |
| recyclable | TEXT CHECK(recyclable IN ('Y','N') | — |

## 参考SQL（SQLite）
```sql
SELECT product_id FROM Products WHERE low_fats = 'Y' AND recyclable = 'Y';
```

## MySQL原始SQL（供方言对比）
```sql
select product_id from Products where low_fats = 'Y' and recyclable = 'Y';
```
