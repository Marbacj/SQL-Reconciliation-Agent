# LeetCode 1581: 进店却未进行过交易的顾客

**难度**: easy
**LeetCode题号**: 1581

## 题目解读
查找进店却未进行过交易的顾客ID以及他们只光顾不交易的次数

## 表: `Visits`

| 字段 | 类型 | 说明 |
|------|------|------|
| visit_id | INTEGER | — |
| customer_id | INTEGER | — |

## 表: `Transactions`

| 字段 | 类型 | 说明 |
|------|------|------|
| transaction_id | INTEGER | — |
| visit_id | INTEGER | — |
| amount | INTEGER | — |

## 参考SQL（SQLite）
```sql
SELECT customer_id, COUNT(visit_id) AS count_no_trans FROM Visits WHERE visit_id NOT IN (SELECT DISTINCT visit_id FROM Transactions) GROUP BY customer_id;
```

## MySQL原始SQL（供方言对比）
```sql
select customer_id, count(visit_id) as count_no_trans from Visits where visit_id not in(select distinct visit_id from Transactions) group by customer_id;
```
